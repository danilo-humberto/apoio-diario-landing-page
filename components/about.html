Sobre o projeto

Um app focado em acessibilidade e apoio para crianças neurodivergentes (TEA, TDAH e deficiências intelectuais).

Objetivo

Melhorar organização de rotinas, previsibilidade e autonomia com tarefas visuais e atividades estruturadas.

Tecnologias

React Native Expo NestJS PostgreSQL AWS Prisma ORM
